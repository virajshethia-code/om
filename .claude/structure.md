# Structure & Organization

## Folder Layout

```
om/
├── CLAUDE.md              # Project overview
├── .claude/               # Project instructions (split)
│   ├── structure.md       # This file — organization rules
│   ├── content.md         # Writing style and format
│   └── roadmap.md         # Learning path and curriculum
├── sanskrit/              # Language lessons
├── philosophy/            # Schools of thought (Vedanta, Yoga, Samkhya, etc.)
├── texts/                 # Notes on primary texts (Vedas, Upanishads, Gita, Epics)
├── practices/             # Meditation, ritual, mantra, yoga
└── connections/           # Modern parallels (AI, cognition, systems)
```

## Naming Conventions

- Lessons: `NN-topic-name.md` (e.g., `01-sanskrit-basics.md`)
- Use lowercase, hyphens for spaces
- Number sequentially within each folder

## File Format

- All content as Markdown
- Include Devanagari script alongside transliteration
- Tables for structured comparisons
- Blockquotes for primary text references
