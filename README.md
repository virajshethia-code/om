# 🕉️ OM — The Darshana Architecture

**Hindu philosophy as a cognitive architecture for AGI.**

Not metaphor. Not spiritual tourism. Engineering specification derived from 5,000 years of systematic inquiry into how information becomes meaning.

## What Is This?

Two things:

### 1. A Complete Hinduism Curriculum (29 Lessons)
Built in a single session using 25 parallel AI agents. Covers:

- **Sanskrit** (6 lessons) — alphabet, sandhi, vibhakti, verb system, numbers, reading shlokas
- **Primary Texts** (12 lessons) — 5 Upanishads (Isha, Kena, Katha, Mandukya, Chandogya), Bhagavad Gita (all 18 chapters), Yoga Sutras, Ramayana, Mahabharata, Puranas

- **Living Traditions** (3 lessons) — Bhakti movement, Tantra & Shakta, Modern Hindu thought
- **Connections** (1 essay) — structural parallels between Hindu philosophy and AI/ML

Every lesson includes Sanskrit in Devanagari + transliteration + root meaning. No New Age repackaging. No shortcuts.

### 2. A Thesis: The Darshana Architecture for AGI
The argument that Hindu philosophy's six schools provide a complete cognitive architecture that current AI approaches lack. See [THESIS.md](THESIS.md).

## The Core Thesis

Current AGI development has no philosophy of mind. The approach is: make it bigger, add more data, hope intelligence emerges.

Hindu philosophy spent 5,000 years building a systematic theory of cognition. The six schools (Shaddarshana) aren't competing theories — they're **complementary reasoning engines**:

| School | Cognitive Function | AGI Application |
|---|---|---|
| **Nyaya** | How do we know what's true? | Epistemic validation, proof chains, fallacy detection |
| **Vaisheshika** | What are the irreducible components? | Atomic decomposition, type systems, ontology |
| **Samkhya** | How do components layer into systems? | Processing pipeline, layer architecture |
| **Yoga** | How do we filter noise and focus? | Attention management, signal/noise optimization |
| **Mimamsa** | How do we extract action from text? | Command interpretation, intent extraction |
| **Vedanta** | How do we find unity beneath contradiction? | Abstraction, contradiction resolution, meta-reasoning |

These aren't metaphors. They're structural isomorphisms — two knowledge systems solving the same problem: how does information become meaning?

## The Architecture

```
PURUSHA         — Meta-awareness ("what am I doing right now?")
BUDDHI          — Fast discrimination + routing
AHAMKARA        — Self-model ("what do I know / not know?")
  ├── MANAS     — Attention routing (meta-attention over attention)
  ├── PRAMANA   — Epistemic tagging (how do I know this?)
  └── GUNA      — Dynamic mode (precision / exploration / retrieval)
SHADDARSHANA    — Six specialized reasoning engines
VRITTI FILTER   — Pre-output classification (valid / hallucination / empty / unknown / memory)
MAYA LAYER      — Representation ≠ reality awareness
KARMA STORE     — Runtime learning from interactions
```

## How This Was Built

25 parallel AI agents. 3 waves. One session.

Each agent received a focused brief and wrote independently — no shared context between agents. Coherence emerged from shared rules (the `.claude/` config files), not central coordination.

This mirrors the Vedic oral tradition: different shakhas (schools) preserved different parts independently across centuries, coherent through shared chandas (meter) and sandhi (rules).

Distributed creation through shared protocol.

## What's Honest

This project takes Hindu philosophy seriously as a knowledge system while being honest about what the tradition got wrong:

**What we keep:** The cognitive architecture, the epistemological framework, the reasoning diversity, the skill lineage (parampara), the compositional language.

**What we drop:** Caste hierarchy, gender restriction, purity/pollution frameworks, ritual as transaction, exclusive claims to authority. The tradition's own reformers — the Bhakti saints, Ambedkar, the Upanishads themselves — already made the case.

**The principle:** Inherited advantage as a launchpad, not a cage. कुलधर्म (family dharma as foundation), never जातिबन्धन (caste as bondage).
